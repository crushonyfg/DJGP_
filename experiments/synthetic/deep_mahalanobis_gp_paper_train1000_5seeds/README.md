# DeepMahalanobisGP Paper Synthetic Train=1000, Five Seeds

This folder contains direct-prediction results for the external
DeepMahalanobisGP implementation on the paper-style L2/LH synthetic
benchmarks, seeds 0-4.

See `docs/experiments_log.md`, entry
`2026-05-13 - deep_mahalanobis_gp_paper_train1000_5seeds`, for the headline
comparison against the existing LMJGP/CEM-EM paper-synthetic table.

Generated by:

```powershell
conda run -n jumpGP python experiments\synthetic\export_paper_synthetic_for_dmgp.py --settings l2_q2_train1000,lh_q5_train1000 --seeds 0,1,2,3,4 --out_dir experiments\synthetic\deep_mahalanobis_gp_paper_train1000_5seeds
conda run -n dmgp_tf1 python experiments\synthetic\run_deep_mahalanobis_gp_paper.py --data_dir experiments\synthetic\deep_mahalanobis_gp_paper_train1000_5seeds --out_dir experiments\synthetic\deep_mahalanobis_gp_paper_train1000_5seeds --settings l2_q2_train1000,lh_q5_train1000 --seeds 0,1,2,3,4 --steps1 300 --steps2 700 --w_samples 0
```

Key files:

- `dmgp_metrics.csv`: per-seed DeepMahalanobisGP metrics.
- `dmgp_aggregate.csv`: five-seed mean/std aggregate.
- `dmgp_summary.json`: run arguments and per-seed rows.
- `dmgp_5seed_run.log`: raw TF1/GPflow log.

Protocol notes:

- X is standardized with train-only statistics.
- DeepMahalanobisGP trains on standardized Y, following the external
  repository's notebook, and metrics are inverse-scaled back to original Y.
- This aggregate intentionally does not run `W` controls or projected-JGP
  diagnostics.
