# DeepMahalanobisGP Paper Synthetic Seed 0

This folder contains a one-seed run of the external DeepMahalanobisGP
implementation on the paper-style L2/LH synthetic benchmarks.

See `docs/experiments_log.md`, entry
`2026-05-13 - deep_mahalanobis_gp_paper_train1000_seed0`, for settings and
headline interpretation.

Generated by:

```powershell
conda run -n jumpGP python experiments\synthetic\export_paper_synthetic_for_dmgp.py --settings l2_q2_train1000,lh_q5_train1000 --seed 0 --out_dir experiments\synthetic\deep_mahalanobis_gp_paper_train1000_seed0
conda run -n dmgp_tf1 python experiments\synthetic\run_deep_mahalanobis_gp_paper.py --data_dir experiments\synthetic\deep_mahalanobis_gp_paper_train1000_seed0 --out_dir experiments\synthetic\deep_mahalanobis_gp_paper_train1000_seed0 --settings l2_q2_train1000,lh_q5_train1000 --seed 0 --steps1 300 --steps2 700 --w_samples 3
conda run -n jumpGP python experiments\synthetic\evaluate_dmgp_w_with_jgp_paper.py --data_dir experiments\synthetic\deep_mahalanobis_gp_paper_train1000_seed0 --settings l2_q2_train1000,lh_q5_train1000 --seed 0
```

Key files:

- `dmgp_metrics.csv`: direct DeepMahalanobisGP predictive metrics.
- `dmgp_w_jgp_metrics.csv`: DeepMahalanobisGP learned `W` evaluated through
  the projected JumpGP-CEM head, plus controls.
- `*_W_mean.npy`, `*_W_var.npy`, `*_W_samples.npy`: test-side projection
  outputs from DeepMahalanobisGP.
- `*_pred_mean.npy`, `*_pred_sigma.npy`: direct DeepMahalanobisGP predictions
  on original Y scale.
- `dmgp_run.log`: raw TF1/GPflow run log.
