# Self-CEM L2/LH train=200/500 5-seed run

See `docs/experiments_log.md`, entry
`selfcem_l2_lh_train200_500_5seeds_20260519`.

This folder fills the missing non-MC self-CEM results for paper-style
synthetic L2/LH settings at train sizes 200 and 500, seeds 0-4.

Primary files:

- `metrics.csv`: per-seed rows.
- `aggregate.csv`: 5-seed aggregate rows.
- `summary.json`: runner metadata.
