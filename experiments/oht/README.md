# OHT Experiments

OHT dataset scripts and related R code.

- `OHTDataset_analysis.py`: OHT analysis harness for JGP/DGP/LMJGP comparisons.
- `OHT_dataset_test.py`: OHT test harness with direct JGP and DJGP paths.

Root `OHTDataset_analysis.py` and `OHT_dataset_test.py` files are compatibility entrypoints.
