# PDEBench-Burgers Projection-Matrix Diagnostics

Generated by `experiments/pdebench/diagnose_projection_matrices.py`.

Files:
- `summary.csv`: one row per seed/method.
- `aggregate.csv`: mean/std over seeds for numeric diagnostics.
- `per_anchor.csv`: per-test-anchor response-geometry diagnostics, including near-threshold flags.
- `summary.json`: nested run metadata and LMJGP posterior diagnostic rows.

Positive `proj_minus_raw_pair_y_abs_corr_median` means projected pairwise distances align
with response differences better than raw-X distances inside local neighborhoods. Negative
`proj_minus_raw_anchor_knn_abs_y_median` means projected nearest neighbors are response-closer
to the anchor than raw-X nearest neighbors.
