# Erosion Experiments

Contains erosion/UCI-style experiment harnesses:

- `erosion_exp_alone.py`: major erosion/LH-style orchestrator.
- `erosion_exp.py`: earlier erosion synthetic benchmark harness.
- `L2_erosion_exp_alone.py`: L2-focused erosion experiment variant.
- `erosion_new_dataset.py`: later K-sensitivity erosion benchmark over UCI-style datasets.

Root `erosion_*.py` and `L2_erosion_exp_alone.py` files are compatibility entrypoints.
