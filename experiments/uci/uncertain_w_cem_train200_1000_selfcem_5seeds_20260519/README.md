# UCI self-CEM train=200/1000 5-seed run

See `docs/experiments_log.md`, entry
`uci_selfcem_train200_1000_5seeds_20260519`.

This folder fills the missing non-MC self-CEM UCI results for train sizes
200 and 1000, seeds 0-4.

Primary files:

- `metrics.csv`: per-dataset, per-train-size, per-seed rows.
- `aggregate.csv`: 5-seed aggregate rows.
- `summary.json`: runner metadata.
