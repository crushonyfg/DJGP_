# Standardized UCI Projection-Matrix Diagnostics

Generated by `experiments/uci/diagnose_projection_matrices_std.py`.

Files:
- `summary.csv`: one row per dataset/seed/method.
- `aggregate.csv`: mean/std over seeds for numeric diagnostics.
- `per_anchor.csv`: response-geometry diagnostics per test anchor.
- `summary.json`: nested run metadata and LMJGP posterior diagnostic rows.

Interpretation notes:
- `feature_eff_count` near the input dimension means the projection spreads weight broadly.
- `metric_anchor_dispersion_rel` near zero means little anchor-specific variation.
- positive `proj_minus_raw_pair_y_abs_corr_median` means projected pairwise distances align
  with response differences better than raw-X distances inside local neighborhoods.
- negative `proj_minus_raw_anchor_knn_abs_y_median` means projected nearest neighbors are
  response-closer to the anchor than raw-X nearest neighbors.
